# misima-anken — 株式会社ミシマ コーポレートサイト

[Peers](https://peers.jp/) を参考にした、**通信業界専門の人材派遣・業務委託会社**のコーポレートサイト（静的サイト）です。
フレームワーク不要の HTML / CSS / JavaScript のみで構成しており、そのまま静的ホスティングに公開できます。

> 注: 社名「株式会社ミシマ」をはじめ、ロゴ・会社概要・沿革・グループ会社名・ニュース等はすべてダミー（プレースホルダ）です。実在の企業ロゴや画像は含んでいません。実データに差し替えてご利用ください。

## サイト構成（サイトマップ）

```
TOP (index.html)
├── 会社情報
│   ├── 代表メッセージ        message.html
│   ├── パーパス・社名の由来   purpose.html
│   ├── 会社概要              profile.html
│   ├── 沿革                  history.html
│   └── グループ会社          group.html
├── サービス
│   └── 事業・サービス一覧     services.html
├── ニュース
│   └── ニュース一覧          news.html
├── 採用情報
│   ├── 採用ページ            recruit.html
│   └── 組織情報              organization.html
└── その他
    ├── お問い合わせ          contact.html
    ├── プライバシーポリシー   privacy.html
    ├── 情報セキュリティ基本方針 security.html
    └── サイトポリシー        sitepolicy.html
```

すべての HTML はリポジトリ直下（フラット構成）に配置しており、相対パスはどの環境でもそのまま動作します。

## ファイル構成

```
.
├── *.html            # 各ページ（上記サイトマップ参照）
├── css/
│   └── style.css     # デザイントークン（CSS変数）＋全スタイル
├── js/
│   └── main.js       # ナビ開閉 / ドロップダウン / スクロール出現 / フォーム
└── assets/
    ├── logo.svg      # ロゴ
    └── favicon.svg   # ファビコン
```

ヘッダー（ドロップダウン付きグローバルナビ）とフッターは全ページ共通です。

## ローカルでの表示方法

リポジトリ直下で簡易サーバーを起動します。

```bash
python3 -m http.server 8000
# ブラウザで http://localhost:8000 を開く
```

## カスタマイズ

- **色 / フォント / 余白**: `css/style.css` 冒頭の `:root` にあるデザイントークン（CSS変数）を変更すると全体に反映されます。ブランドカラーは `--brand` / `--brand-dark`。
- **テキスト / 会社情報 / ニュース**: 各 `*.html` 内のダミーテキストを実際の内容に差し替えてください。
- **ロゴ**: `assets/logo.svg` / `assets/favicon.svg` を差し替えてください。
- **ナビ / フッター**: 全ページに同じマークアップが埋め込まれています。リンク構成を変更する場合は各ページのヘッダー／フッターを揃えて編集してください。
- **お問い合わせフォーム**（`contact.html`）: 現在は静的なデモ動作です。実送信にはバックエンド（フォーム送信サービス等）への接続が必要です。

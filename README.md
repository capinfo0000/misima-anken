# misima-anken — 株式会社ミシマ コーポレートサイト

[Peers](https://peers.jp/) を参考にした、企業型コーポレートサイト（静的サイト）です。
フレームワーク不要の HTML / CSS / JavaScript のみで構成しており、そのまま静的ホスティングに公開できます。

## ファイル構成

```
.
├── index.html        # トップページ一式（ワンページ構成）
├── css/
│   └── style.css     # デザイントークン（CSS変数）＋全スタイル
├── js/
│   └── main.js       # ナビ開閉 / スムーススクロール / スクロール出現 / フォーム
└── assets/
    ├── logo.svg      # ロゴ
    └── favicon.svg   # ファビコン
```

## セクション構成

ヘッダー / ヒーロー / パーパス / 代表メッセージ / サービス / グループ会社 / ニュース / 採用バナー / お問い合わせ / フッター

## ローカルでの表示方法

リポジトリ直下で簡易サーバーを起動します。

```bash
python3 -m http.server 8000
# ブラウザで http://localhost:8000 を開く
```

`index.html` をブラウザで直接開いても表示できます。

## カスタマイズ

- **色 / フォント / 余白**: `css/style.css` 冒頭の `:root` にあるデザイントークン（CSS変数）を変更すると全体に反映されます。ブランドカラーは `--brand` / `--brand-dark`。
- **テキスト / 会社名 / ニュース**: `index.html` 内のダミーテキストを実際の内容に差し替えてください。
- **ロゴ**: `assets/logo.svg` / `assets/favicon.svg` を差し替えてください。
- **お問い合わせフォーム**: 現在は静的なデモ動作です。実送信にはバックエンド（フォーム送信サービス等）への接続が必要です。

> 注: 掲載しているロゴ・社名・テキストはすべてダミー（プレースホルダ）です。実在の企業ロゴや画像は含んでいません。
